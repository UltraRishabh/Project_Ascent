# Project Ascent — 3D Platformer Prototype

A physics-based 3D platformer built in Godot.

## Features
- Priority-based Finite State Machine managing 7+ traversal states
- 3-point raycast sensing for ledge detection
- Ledge clambering, wall climbing, double jump, dash
- Jump buffering (120ms) and coyote time
- Asymmetric gravity and variable jump height

## Playable Demo
https://ultralyn.itch.io/ascent-3d-platformer

- Godot 4
- GDScript
